## Features

1. Single Page Application
2. Inputs Validators
3. Add, delete, and edit new courses, articles, users, etc.
4. Dark & light mode

## Installation

Use npm to install Redux CMS.

run Frontend :
```bash
cd Front-End
npm i
npm run dev
```
